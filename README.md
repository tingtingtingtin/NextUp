# NextUp
`This project is currently in development, and it exists because I wanted to learn Kotlin.`

An ADHD-focused task manager built for Android and Wear OS. This app uses the 1-3-5 daily productivity rule to limit executive function load.

## Tech Stack
* Kotlin
* Jetpack Compose
* Material 3
* Compose for Wear OS

## Target Architecture
* Android App (SDK 30+)
* Wear OS (SDK 33+)

## Features
- **Daily Reset:** Migration logic that snoozes unfinished tasks and clears completed ones.
- **Priority Slots:** Fixed 1-3-5 structure to prevent overwhelming lists.
- **Recurring Tasks:** One-tap toggle for daily habits.
- **Wear OS Integration**: Includes tile and complication support
- **Cross-Device Sync:** (Planned) Wearable Data Layer for Phone-to-Watch parity.
